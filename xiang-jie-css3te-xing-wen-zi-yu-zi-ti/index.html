详解CSS3特性----文字与字体

  • text-overflow：用来设置是否使用一个省略标记（...）标示对象内文本的溢出。

    • text-overflow: clipboard | ellipsis(clipboard代表剪切，ellipsis代表显示省略标记)

    • 但是text-overflow只是用来说明文字溢出时用什么方式显示，要实现溢出时产生省略号的效果，还须定义强制文本在一行内显示（white-space:nowrap）及溢出内容为隐藏（overflow:hidden），只有这样才能实现溢出文本显示省略号的效果，代码如下：


text-overflow:ellipsis; 
overflow:hidden; 
white-space:nowrap;

  • word-wrap:

    • word-wrap:normal | break-word(normal代表控制连续文本换行，break-word表示内容将在边界换行)

  • @font-face:

    • @font-face能够加载服务器端的字体文件，让浏览器端可以显示用户电脑里没有安装的字体。

@font-face {
    font-family : 字体名称;
    src : 字体文件在服务器上的相对或绝对路径;
}

  • p必须设置font-family:设置@font-face中font-family同样的值

  • text-shadow：

    • 语法：text-shadow: X-Offset Y-Offset blur color;
