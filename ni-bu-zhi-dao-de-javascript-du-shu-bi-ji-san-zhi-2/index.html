《你不知道的JavaScript》读书笔记（四）----原生函数

  • 通过构造函数创建出来的是封装了基本类型值的封装对象

  • 所有typeof为"Object"的对象都包含一个内部属性[[class]]，它无法直接访问，可以使用Object.prototype.toString()来查看

  • 当我们使用.length属性操作基本类型值时，JavaScript会自动为基本类型值包装一个封装对象

  • **注意！**以下代码中a是对象是真值


  var a = new Boolean( false );
  if (!a) {
  	console.log( "Oops" ); // 不会执行
  }

  • 可以使用valueOf()来拆封 基本对象封装对象，有时也会发生隐式拆封

  • 当Array()构造函数只带一个数字参数的时候，该参数会被当做数组的length

  • var a = [ , , , ] a.length // 3 而不是 4因为它在末尾多加了一个逗号

  • 构造函数生成的空数组 可以执行join操作，却不能执行map (因为数组中不存在任何单元)

  • Date(...)见P41，创建错误对象主要是为了获得当前运行栈的上下文(.stack属性访问)，还有message属性

  • Symbol 主要用于私有或特殊属性防止对象属性重名

  • 如下所见Function.prototype是一个函数，RegExp.prototype是一个正则表达式，Array.prototype是一个数组

    
typeof Function.prototype;			// "function"
Function.prototype();				// 它是一个空函数!

    RegExp.prototype.toString();		// "/(?:)/" -- 空的正则表达式
    
"abc".match( RegExp.prototype );	// [""]
