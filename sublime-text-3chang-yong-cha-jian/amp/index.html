Sublime Text 3常用插件

by jeekdong

2017-05-24

Sublime Text：一款具有代码高亮、语法提示、自动完成且反应快速的编辑器软件，不仅具有华丽的界面，还支持插件扩展机制

使用Sublime Text，插件则是不可缺少的存在。而插件的备份就显得非常的重要，否则每次更换设备就需要重新手动安转插件，显得十分繁琐。在sublime 中这件事很简单，只需将Packages（Preferences > Browse Packages...）中内容拷贝一份，同步云端即可；至于存储何处，云盘，Github，都可以，譬如先前有存一份于sublime_packages，每有所需只要 下载下来，替换掉原Packages下内容就好。

1. 安装package control

Sublime Text的强大就是她拥有非常强大的扩展性。我们可以根据自己的需要安装不同的插件，而package control就是用来管理插件的插件。

首先打开package control的官方网站。复制下面这一段代码:

import urllib.request,os,hashlib; h = 'df21e130d211cfc94d9b0905775a7c0f' + '1e3d39e33b79698005270310898eea76'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

通过View–>Show console或使用快捷键：Ctrl+~开控制台窗口，粘贴上面的代码，回车，然后就是等待安装了，需要一定的时间。安装完成后重启。


如果安装成功，就可以在Preferences菜单下看到Package Settings和Package Control两个菜单。

2. package control使用方法

在Sublime Text 3中按下快捷键Ctrl+Shift+P在出现的文本框中输入Install Package(或直接输入“ip”)选中Install Package并回车

输入或选择你需要的插件再按Enter（回车）就可以安装插件了

安装过程中左下角会有提示，可以耐心等待

3. 插件推荐

3.1 Emmet插件

  • Emmet插件可以说是使用Sublime Text进行前端开发必不可少的插件,它让编写HTML代码变得极其简单高效

  • 官方网站

  • 中文文档：地址

  • 前端开发必备！Emmet使用手册

  • 速记表

3.2 AdvancedNewFile：快速新建文件
  • 假设有文件夹file。我们正在输入代码，又想在新的子目录下新建html文件的话用传统方式得很多步，新建目录，新建文件，保存等等等。
  • 但是有了该插件之后，事情就变得简单了许多，只需要按下Ctrl+alt+N，输入文件夹以及文件名，你就会看到如下效果:（回车，你会发现已经子目录下的文件已经新建完成了！）
3.3 JsFormat插件
  • 这是一款将JS格式化的插件
  • 同样使用Package Control安装JsFormat插件后即可在JS文件中通过鼠标右键->JsFormat或键盘快捷键Ctrl+Alt+F对JS进行格式化
3.3 Sidebar Enhancements：增强侧边栏
  • SideBarEnhancements是一款很实用的右键菜单增强插件
  • 在安装该插件前，在Sublime Text左侧FOLDERS栏中点击右键
    只有寥寥几个简单的功能
  • 通过Package Control安装SideBarEnhancements插件后,可以看到，文件夹栏右键菜单马上增强不少
  • 可以在浏览器中打开，配置不同文件的打开方式，十分方便。
3.4 CssComb插件
  • CssComb是为CSS属性进行排序和格式化插件 官网
  • 使用Package Control安装CssComb插件要注意，它依赖于Node.js，所以你要先安装Node.js才能运行该插件
3.5 Color Highlighter ：CSS颜色高亮
  • 这个插件我等了很久了，我最早用Sublime Text写CSS时候就在想“这堆颜色码谁知道是什么颜色”。
  • Color Highlighter这个插件会检测CSS文件中的颜色码，不论是Hex码或者RGB码都能很好的显示。
  • Color Highlighter能够设置成用背景色或者边框提示颜色，我一般在Settings里做这样的设置：
{
  "ha_style": "filled",
  "icons": false
}
```

######3.6 CSS3:CSS3语法高亮

* 默认安装的Sublime Text对CSS3的支持让人抓狂，帧动画？别开玩笑了你只会看到一片白色的纯文本一样的代码。

###3. 插件的管理
 ---

当我们安装了很多插件的时候，如何查看sublime安装了哪些插件呢？

按`ctrl`+`shift`+`p`，输入`package`，选择`list packages`，就看到了。
或者直接查看`Installed Packages`目录。
![](/content/images/2017/05/QQ--20170524214458.png)
如何删除我们不想要的插件呢？

`Ctrl`+`Shift`+`P`调出命令面板，输入`remove`，调出`Remove Package`选项并回车，选择要删除的插件即可，当然，更新插件，`upgrade packages`，通过简单的几个命令就可以方便的管理我们的插件了
